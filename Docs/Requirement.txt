PokeDex AI platform

Core features

- Sign up
- Login

- OAuth Google (optional)

2. Pokemon search
- Search by name
- Search by type
- Search by generation
- Pagination
- Infinite scroll

- Stats
- Abilities
- Types
- Moves
- Evolution chain
- Weakness/resistance

- Remove favorites

    + Reports
